# PasswordStrengthChecker 🔐

A simple Java program that checks the strength of a user's password based on length and character types.

## 📌 Features
- Checks for:
  - Length of password
  - Uppercase and lowercase letters
  - Numbers
  - Special characters
- Rates password as **Weak**, **Medium**, or **Strong**
